## Metadata (Next.js Metadata API)

Реализовано через `app/layout.tsx` (global) и `export const metadata` на каждой странице.

### Пример (главная)
```typescript
title: 'Контейнерный терминал в Новороссийске — хранение и перетарка грузов | Терминал Приморский'
description: 'Терминал Приморский: хранение контейнеров и грузов, перетарка, обработка и перевозка рядом с портом Новороссийска. Отправьте параметры и получите расчёт.'
canonical: https://terminal-primorskiy.ru/
```

### Вспомогательные функции
- `lib/seo/index.ts` — `buildMetadata()`, `buildServiceMetadata()`
- `lib/seo/jsonld.ts` — JSON-LD хелперы

## Structured Data (JSON-LD)

| Тип | Страницы | Файл |
|---|---|---|
| `Organization` | Главная | `lib/seo/jsonld.ts` |
| `LocalBusiness` | Главная, Контакты | `lib/seo/jsonld.ts` |
| `Service` | Все страницы услуг | `lib/seo/jsonld.ts` |
| `BreadcrumbList` | Все внутренние страницы | `lib/seo/jsonld.ts` |
| `FAQPage` | Главная, FAQ, страницы услуг | `lib/seo/jsonld.ts` |

> **Важно:** `LocalBusiness` отображается только если `company.address` заполнен в `config/company.ts`.
> `AggregateRating` не добавлен — нет верифицированных отзывов.

## Локальное SEO

### NAP (Name, Address, Phone)
Данные централизованы в `config/company.ts`. Отображаются на всех страницах (футер, контакты) единообразно.

**⚠️ Перед публикацией:**  
Убедитесь, что адрес и телефон одинаковы:
- На сайте (config/company.ts)
- В Яндекс Бизнесе
- В 2ГИС
- В Google Business Profile
- В других справочниках

### Зоны обслуживания
Указаны на странице `/kontakty`: Новороссийск, Краснодарский край, ЮФО.

## Подключение Search Console и Яндекс Вебмастера

### Яндекс Вебмастер
1. Перейти на https://webmaster.yandex.ru
2. Добавить сайт
3. Подтвердить через HTML-метатег (добавить в `app/layout.tsx` в metadata)
4. Отправить `sitemap.xml`

### Google Search Console
1. Перейти на https://search.google.com/search-console
2. Добавить ресурс (тип: URL-prefix)
3. Подтвердить через HTML-метатег или файл
4. Отправить `sitemap.xml`

### Добавление метатегов подтверждения
В `app/layout.tsx` добавить в `metadata`:
```typescript
verification: {
  yandex: 'ВАШИ_ДАННЫЕ',  // из Яндекс Вебмастера
  google: 'ВАШИ_ДАННЫЕ',  // из Google Search Console
},
```
